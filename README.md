### h3 Инструкция 

 1. Запустить скрипт CreateDatabase.sql 1
 2. Запустить скрипт InitialData.sql 2
 3. Установить .NET Core SDK, если не установлен на компьютере 
 (https://www.microsoft.com/net/download/windows) 3
 4. Открыть  JobChart.sln из папки Services в Visual Studio 2017.
  Убедиться, что JobChart выбран стартовым проектом. Запустить по F5. 4
 5. Проверить http://localhost:63342/api/chart 5
 6. Установить .NET Core SDK, если не установлен на компьютере (https://www.microsoft.com/net/download/windows) 6
 7. Открыть Angular веб сайт из папки jobchartapp в Visual Studio Code 7
 8. Выполнить команду npm install в папке jobchartapp 8
 9. Выполнить команду npm start в папке jobchartapp 9
 10. Открыть http://localhost:4200/ 10
 11. Нажать на кнопку Load Chart Data 11

3. База данных: SQL Server 2017 3
