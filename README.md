Инструкция

1)запустить скрипт CreateDatabase.sql

2)запустить скрипт InitialData.sql

3)Установить .NET Core SDK, если не установлен на компьютере 
(https://www.microsoft.com/net/download/windows)

4)открыть  JobChart.sln из папки Services в Visual Studio 2017.
 Убедиться, что JobChart выбран стартовым проектом. Запустить по F5.

5)проверить http://localhost:63342/api/chart

6)Установить .NET Core SDK, если не установлен на компьютере (https://www.microsoft.com/net/download/windows)

7)открыть Angular веб сайт из папки jobchartapp в Visual Studio Code

8)выполнить команду npm install в папке jobchartapp

9)выполнить команду npm start в папке jobchartapp

10)открыть http://localhost:4200/

11)нажать на кнопку Load Chart Data

Технологии

Вебсайт: TypeScript, Angular, Highcharts, Visual Studio Code

Сервисы: C#, Entity Framework Core, .NET Core 2.0, Visual Studio 2017

База данных: SQL Server 2017

Система контроля версий: Git
